# Players por Segmento — Ponto de Partida

Este arquivo contém players conhecidos por segmento para iniciar a pesquisa.
**Não substitui a pesquisa real** — use como ponto de partida para saber onde olhar.
Sempre verifique se os players ainda existem e se há novos entrantes relevantes.

## Financeiro para PME (Brasil)

**Líderes:**
- Conta Azul — financeiro + fiscal, foco PME
