# Perguntas de Entrevista — Ferramenta Interna

Use em rodadas de no máximo 4 perguntas cada.
Pule perguntas já respondidas pelo contexto existente.

---

## Rodada 1 — O problema

1. Qual tarefa você faz hoje que toma mais tempo do que deveria? Descreva o processo atual.
2. Quantas pessoas fazem essa tarefa? Com que frequência?
3. Qual é o custo real do processo atual? (tempo perdido, erros, retrabalho)
4. O que já foi tentado para resolver esse problema antes?

## Rodada 2 — Escopo

1. Se o sistema fizesse apenas uma coisa, qual seria?
2. Quais etapas do processo atual o sistema vai substituir?
3. Quais etapas vão continuar sendo manuais mesmo com o sistema?
4. Tem alguma funcionalidade que parece útil mas pode esperar?

## Rodada 3 — Usuários

1. Quem vai usar o sistema? Liste as pessoas pelo papel, não pelo nome.
2. Alguém de fora da equipe vai precisar acessar? (cliente, fornecedor, auditor)
3. Todos os usuários veem as mesmas informações ou tem restrições?
4. Quem vai ser o responsável por manter o sistema funcionando?

## Rodada 4 — Dados e integrações

1. De onde vêm os dados que o sistema vai processar? (planilha, outro sistema, entrada manual)
2. O resultado precisa sair em algum formato específico? (relatório, planilha, email)
3. Precisa se conectar com algum sistema que já existe na empresa?
4. Existe dado sensível que precisa de proteção especial?

## Rodada 5 — Critérios de sucesso

1. Como você vai saber que o sistema está funcionando bem?
2. Qual é o pior que pode acontecer se o sistema falhar?
3. Tem prazo para o sistema estar funcionando?
4. Quem aprova se o sistema está pronto para uso?
